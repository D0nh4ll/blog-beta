# 欢迎来到我的技术博客

这是我使用 Jekyll 搭建的个人博客的第一篇文章。在这个博客中，我计划分享一些关于前端开发、Web 技术以及个人学习心得的内容。

## 为什么选择 Jekyll？

Jekyll 是一个简单而强大的静态网站生成器，它有以下优点：

1. 使用 Markdown 写作，简单直观
2. 支持代码高亮
3. 可以轻松部署到 GitHub Pages
4. 完全可定制的主题和布局
